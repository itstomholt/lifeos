Sync Setup — One-Time

  1. Create a private GitHub repo (or use your vault repo).
  2. GitHub → Settings → Developer settings → Fine-grained tokens → Generate. Scope to only that repo; permission: Contents — Read and write.
  3. Enter repo, branch, folder, token above → Connect & Test.
  4. Optional: Obsidian Git plugin pointed at the same repo.
  5. Repeat on each device with the same settings.
