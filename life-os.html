Vault Sync (GitHub)

Syncs two files to a repo: life-os/state.json (app source of truth) and life-os/basic-living.md (Obsidian-readable). Point Obsidian Git at the same repo and the vault stays current. Token is stored on this device only — never written to the repo.

Sync Setup — One-Time

  1. Create a private GitHub repo (or use your existing vault repo).
  2. GitHub → Settings → Developer settings → Fine-grained tokens → Generate. Scope it to only that repo, permission: Contents — Read and write. Nothing else.
  3. Paste repo, branch, folder, and token above → Connect & Test.
  4. In Obsidian, install the Git community plugin pointed at the same repo. The markdown file appears in your vault and updates on every sync.
  5. Repeat step 3 on each device (phone, computer) with the same settings.

Sync model: newest wins by timestamp. The app pulls on open and pushes ~5 s after any change. Edits made to the markdown in Obsidian are for reading/archiving — the JSON is the app's source of truth.
